Block-Based

This method depends on dividing the image into blocks of different shapes, including squares and circles that can be non-overlapping or overlapping to be used afterwards in the preprocessing stage.

keypoint-based

Keypoint-based method is non block-based. Key point features are extracted from the distinct area such as edges and corners. Each feature is represented by a set of descriptors. Then, Descriptors and features are collected to be classified and then matched to identify duplicated regions in the image.
